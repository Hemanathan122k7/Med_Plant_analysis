# Medicinal Plant Identifier Icons

This directory contains icon files for the application.

## Icon Requirements

- Format: PNG or SVG
- Recommended sizes: 16x16, 32x32, 64x64, 128x128
- Transparent background for non-square icons

## Icon List

- `logo.png` - Application logo
- `search.svg` - Search icon
- `camera.svg` - Camera icon
- `filter.svg` - Filter icon
- `favorite.svg` - Favorite/heart icon
- `compare.svg` - Compare icon
- `share.svg` - Share icon
- `leaf.svg` - Leaf/plant icon
- `warning.svg` - Warning icon
- `info.svg` - Information icon

## Usage

Icons can be referenced in CSS or HTML:

```html
<img src="assets/images/icons/logo.png" alt="MediPlant Logo">
```

```css
.icon {
    background-image: url('../../assets/images/icons/leaf.svg');
}
```
