# Plant Images

This directory contains images of medicinal plants featured in the application.

## Image Requirements

- Format: JPG, PNG, or WEBP
- Recommended size: 800x600 pixels (minimum)
- Aspect ratio: 4:3 or 16:9
- File size: Keep under 500KB per image
- Naming convention: `plant-name.jpg` (lowercase, hyphen-separated)

## Current Plants

Add images for the following plants:

1. `aloe-vera.jpg` - Aloe Vera plant
2. `turmeric.jpg` - Turmeric root and plant
3. `peppermint.jpg` - Peppermint leaves
4. `lavender.jpg` - Lavender flowers
5. `ginger.jpg` - Ginger root
6. `chamomile.jpg` - Chamomile flowers

## Image Sources

Images should be:
- High quality and well-lit
- Clearly showing the plant's identifying features
- Royalty-free or properly licensed
- Properly credited if required

## Attribution

If using third-party images, maintain an `attributions.txt` file with:
- Image filename
- Source URL
- License type
- Attribution requirements
